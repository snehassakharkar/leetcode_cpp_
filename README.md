# LeetCode Beginner Practice (C++)
This repository contains some beginner problems that I solved from the LeetCode Beginner Guide.  
The purpose of this assignment is to start practicing coding and improve problem-solving skills.

## Problems
**1. Running Sum of 1D Array**  
In this problem I calculated the running sum of the array elements.  
Example: Input → 1 2 3 4  
Output → 1 3 6 10

**2. FizzBuzz**  
This program prints numbers from 1 to n.  
If a number is divisible by 3 it prints "Fizz",  
if divisible by 5 it prints "Buzz",  
and if divisible by both it prints "FizzBuzz".

**3. Number of Steps to Reduce a Number to Zero**  
In this problem, if the number is even we divide it by 2,  
and if it is odd we subtract 1.  
We repeat the process until the number becomes 0 and count the steps.
